    Controls

    • click / drag while in build mode to create tiles
    • click / drag while in paint mode to colour tiles
    • up and down arrow keys while in build mode to change tile height
    • w, a, s, d to move the camera
    • scroll to zoom in / out
    • q to quickly change the paint colour of a tile in build or paint mode
